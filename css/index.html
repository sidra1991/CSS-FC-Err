appunti css

con css si intende un linguaggio che permette di modificare i valori della macchina che ne va a leggere il contenuto dando cosi possibilità di modificarne style e aspetto dello stesso.
le necessita al fine della sua funzione sono poche, basta inserire un link direttamente nel head del file html per permetere il suo funzionamento in tale file, basta quindi inserire le coordinate del file e cosi il file css sarà letto con il file html dalla macchina.

selettori

per attribuire una funzione ad un oggetto basta indicarla nel file css usando gli appositi selettori.

  • *=usando l'asterisco si selezionano tutte le TAG del file html
  • TAG= inserendo una specifica TAG si selezioneranno tutte le TAG di quel tipo
  • .classe= con il punto si seleziona una classe specifica indicata subito dopo il punto
  • #id= con il cancelletto si seleziona una id specifica indicata subito dopo il cancelleto
  • TAG.class= in questo modo si seleziona una tag che abbia anche una specifica classe
  • TAG,.class=la virgola permette di chiamare più oggetti assieme permettendo di selezionare piu oggetti anche indipendenti l'uno dal altro.
  • TAG TAG= usando lo spazio indica la selezione del secondo oggetto indicato che sia discendente dal primo.
  • TAG>TAG= usando il maggiore indica di voler selezionare il secongo oggetto indicato che sia direttamente figlio del primo.
  • TAG:before e TAG:after=permette di inserire del contenuto come primo oggetto o come ultimo oggetto negli oggetti indicati.
  • TAG[type="type"]=permette di selezionare un determinato oggetto che sia in possesso di un determinato attributo.
  • TAG:hover=permette di determinare il comportamento di un oggetto quando viene al contatto con il mouse nel layout
  • TAG:visited=permette di determinare il comportamento dopo che il mouse ha applicato il click sul oggetto
  • TAG:frist-cild o TAG:last-cild=permettono di selezionare il primo oppure l'ultimo figlio di un oggetto
  • TAG+= indica i TAG immediatamente successivi a quello indicato.

dopo i selettori le indicazioni devono essere inserite tra parentesi graffe {} i comandi sono divisi dalle funzioni dai 2 punti (:), per chiudere un comando serve il punto e virgola che funziona anche da divisorio tra un comando e un alrtro(;)

molti oggetti non possono essere affiancati, mentre altri non possono avere delle dimenzioni specifiche, la funzione display vuole cambiare questi comportamenti di defoult la dove necessario.

comportamenti di defoult

  • inline sono gli oggetti che di defoult possono essere affiancati tra loro, ma questi non possono ottenere delle dimenzioni stabilite.

  • block sono gli oggetti che possiedono lo spazzio di tutta la riga per cui non possono essere affiancati l'un altro, possono avere dimenzioni però specifiche.

per cambiare questi comportamenti di defoult bisogna usare il selettore e tra le parentesi indicare la regola display: seguito dal comportamento che ci serve inline, block, inline-block oppure none .

dimenzioni

stabilire una dimenzione ad un oggetto ha funzioni esclusive del layout non visibili in progettazione ma letti esclusivamente dalla macchina ma se privo di colore sara visibile soltanto con l'ispettor.

esempio

per impostare delle dimenzioni è sufficiente usa re 2 comandi:

  • whidth:= per impostare la larghezza del oggetto
  • height:= per impostare l'altezza del oggetto

le unita di misura usate per impostare le dimenzioni sono diverse e si possono usare :

  • px = è sufficiente impostare la grandezza desiderata esprimendola in pixel (whidth: 60px;)
  • % = usando le percentuali si ottiene un oggetto con magior proprietà responsive, le percentuali indicano sempre la percentuale di grandezza del padre (whidth:30%; height:30% figlio del esempio precedente )
  • vh e vw =sono unita di misura rapportate alle dimenzioni dello schermo del dispositivo usato (whidth:10 wh; height:10vw; )

MARGIN

il margin è il margine di distanza tra un oggetto e l'altro allinfuori del bordo stabilito.
il margine di defoultviene calcolato dalla macchina, mentre puo essere anche stabilito manualmente con il comando e quale margine vogliamo affrontare (margin-top:)
nel comando possiamo stabilire le dimenzioni con tutte le unita di misura.
è inoltre possibile stabilire tutti e 4 i margini esprimendo un unico valore (margin:10px) oppure esprimendoli tutti e 4 nel seguente ordine top,left,left,right (margin:5px 5px 5px 5px; )
si possono usare anche valori negativi o azzerarlo

position

tutti gli oggetti che siano blck o inline-bock possono essere spostati con il comando position: dando poi indicazioni sullo spostamento mendiante top-left che indicano lo spostamento dalla posizione attuale. esistono divesi tipi di position

  • relative: permette di spostare un oggetto relativamente al padre.
    position: relative;
    top: 10px;
    left : 10px
  • absolute: permette di spostare un oggetto togliendolo dal flusso di dati del layout, questo quidni lo rende anche indipendente dal padre e permette il suo posizionamento anche al esterno di esso.
    position: absolute (quadrato nero);
    top: 3687px;
    left : 400px
    il quadratino nero èra andado a inizio pagina per questo il top e il left sono cosi alti.
  • fixed: gli oggetti fixed sono gli oggetti fissi che non subiscono gli effetti di scrol della pagina. (il quadratino nero in alto a destra che ti segue ovunque.)
